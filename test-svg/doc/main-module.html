initalizeTests(tests)

source code 

Initalise la structure de tests pour faciliter l'utilisation après. Création d'identifiants uniques pour les filtres, les tests etc.

launchPreprocessing(tests)

source code 

Lance la serie des pretraitements tels que definis dans la stucture `tests'. On charge et execute en série les filtres, le nom du fichier résultat est dans le champ `output' de partie preprocessing de la structure `tests'.

launchCapture(tests)

source code 

Lance les captures. `tests' est la grande structure de tests. Le résultat de chaque capture est stocké dans capture['output']

launchPostprocessing(tests)

source code 

Lance la serie des posttraitements tels que definis dans la stucture `tests'. On charge et execute en série les filtres, le nom du fichier résultat est dans le champ `output' de partie preprocessing de la structure `tests'.

buildComparisons(tests)

source code 

Pour chaque test, construit les instances de comparaisons, en croisant les résultats si plusieurs navigateurs ont été lancés par exemple. On met à jour les test['comparison'].

launchDiagnostic(tests)

source code 

Lance les diagnostics. `tests' est la grande structure de tests. Pour chaque comparaison, on lance le module de diagnostic défini dans la structure.

init()

source code 

Initialise la configuration du programme. En particulier, on règle les repertoires où seront sauvés les résultats intermédiaires.
